# START@WORK v0.8.1 deployen op GitHub Pages

## Belangrijk
Upload de inhoud van deze map naar de root van je repository. Dus `index.html` moet rechtstreeks zichtbaar zijn in de repository.

Goed:
- index.html
- styles.css
- app.js
- assets/
- downloads/
- data/
- manifest.webmanifest
- favicon.svg
- 404.html

Fout:
- startatwork_release_v08_1/index.html

## Deploy
Als GitHub Pages al actief is:
1. Upload/vervang de bestanden.
2. Klik op **Commit changes**.
3. Wacht 1 tot 5 minuten.
4. Vernieuw met Ctrl+F5 of open in een privévenster.

## Instellingen
Settings → Pages:
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

## Test na publicatie
Controleer:
- de startpagina opent;
- Smartschool-, Outlook-, Hast- en NBN-logo laden;
- NBN-PDF opent;
- Downloads werken;
- Leerplandoelen verschijnen.
