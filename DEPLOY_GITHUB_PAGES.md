# Deploy START@WORK Platform

Upload de inhoud van deze map naar de root van je repository.

Correct:
- index.html
- styles.css
- app.js
- assets/
- downloads/
- data/
- favicon.svg
- manifest.webmanifest
- 404.html

Settings → Pages:
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

Na uploaden: Commit changes. Wacht enkele minuten.
